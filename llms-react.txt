# Phosphor Icons - React

> A flexible icon family for React applications with 1,248+ icons in 6 weights: Thin, Light, Regular, Bold, Fill, and Duotone. Tree-shakable components with powerful API for styling.

Phosphor Icons is a comprehensive icon library designed at 16x16px for perfect clarity at small sizes while scaling beautifully for larger displays. Each icon retains raw stroke information for fine-tuning and comes as a fully customizable React component.

## Installation & Setup

Install the package via npm or yarn:
```bash
npm install @phosphor-icons/react
# or
yarn add @phosphor-icons/react
```

## Basic Usage

Import icons individually for tree-shaking optimization:
```jsx
import { Heart, Smiley, Horse } from "@phosphor-icons/react";

function App() {
  return (
    <div>
      <Heart size={32} color="hotpink" weight="fill" />
      <Smiley size={24} weight="duotone" />
      <Horse weight="thin" />
    </div>
  );
}
```

## Icon Props

All Phosphor React components accept the following props:

- **color** (string): Icon color, any valid CSS color value. Default: "currentColor"
- **size** (number | string): Icon size in pixels. Default: 16
- **weight** (string): Icon weight/style - "thin", "light", "regular", "bold", "fill", or "duotone". Default: "regular"
- **mirrored** (boolean): Flip icon horizontally. Default: false
- **alt** (string): Accessibility text for screen readers
- Plus all standard SVG element props (className, onClick, style, etc.)

## Weight Styles

Each icon comes in 6 distinct weights:

- **Thin**: Delicate 1px strokes for minimal, elegant designs
- **Light**: Refined 1.5px strokes for subtle, modern interfaces
- **Regular**: Balanced 2px strokes, the default weight
- **Bold**: Strong 2.5px strokes for emphasis and impact
- **Fill**: Solid filled icons for maximum visual weight
- **Duotone**: Two-tone icons with 20% opacity secondary layer for depth

## Context API for Global Styling

Set default props for all icons in your app:
```jsx
import { IconContext } from "@phosphor-icons/react";

function App() {
  return (
    <IconContext.Provider value={{
      color: "limegreen",
      size: 32,
      weight: "bold",
      mirrored: false
    }}>
      {/* All icons here inherit these defaults */}
    </IconContext.Provider>
  );
}
```

## Advanced Usage

Icons are transparent SVG wrappers, supporting all SVG props and event handlers:
```jsx
<Heart
  className="hover-icon"
  onClick={() => console.log("Clicked!")}
  style={{ cursor: "pointer", transition: "all 0.2s" }}
  onMouseEnter={(e) => e.target.style.color = "red"}
  onMouseLeave={(e) => e.target.style.color = "black"}
/>
```
